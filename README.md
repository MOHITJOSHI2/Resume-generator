# Resume Generator with arguments 
* This is a simple resume generator javascript code that uses ".json" file for resume information and create a ".pdf" file as resume.

## How Does it work?
* Firstly clone the git repo in a directory in your computer
* After cloning the repo run "npm i" in your terminal. (It will download all necessary library used in the program"
* Then for executing the program you require a ".json" file with all your inforamtion. (Check the resumeInfo.json file for reference).
* This program will create a ".pdf" file as output. (You will have to give name to the ".pdf" file yourself)
* Then finally in terminal run "node resumeGenerator.js <your .pdf output file name> <Your .json file name>" and hit enter
* It will create a ".pdf" resume in your directory.
